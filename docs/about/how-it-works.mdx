Encord's Python Software Development Kit (SDK) helps you program mobile applications in one single place. Python SDK enables the user to write codes that manage Oracle Cloud Idenfitier (OCI) resources. The primary purpose of the SDK is to provide an easy-to-use Python environment enabling developers to authenticate a user, retrieve data, and upload data/results.
